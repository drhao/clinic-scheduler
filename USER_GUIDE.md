# 門診排班系統 - 使用者指南 (User Guide)

歡迎使用門診排班系統！本指南將協助您輕鬆管理每週三的門診值班表。

## 1. 開始使用
請直接使用網頁瀏覽器 (Chrome, Edge, Safari 等) 開啟 **`index.html`** 檔案即可使用，無需安裝任何軟體。

## 2. 人員管理 (Manage Staff)
您可以在畫面右側的 **"Manage Staff"** 區塊管理醫生或員工名單。

-   **新增人員**: 輸入姓名並設定 "Max" (每月值班上限)。點擊 **Add** 新增。
-   **編輯 (Edit)**: 點擊姓名旁的 "Edit" 按鈕可修改姓名或上限。
-   **刪除 (Delete)**: 點擊 "Delete" 可移除該人員。*注意：這也會一併移除該人員已排定的值班紀錄。*

## 3. 設定假期 (Holidays)
若遇到國定假日或特定週三不需看診：
1.  在左側日曆中找到該日期。
2.  勾選格子內的 **"Holiday"** 核取方塊。
3.  系統會顯示 "No Duty" (無值班)，且**不會**安排任何人於該日值班。

## 4. 設定不便時間 (Constraints)
若某位同仁當天無法值班 (例如請假)：
1.  前往右側的 **"Manage Constraints"** 區塊。
2.  **Select User**: 選擇該位同仁。
3.  **Date**: 選擇日期。
4.  **Time**: 勾選 "Wednesday AM" (上午) 或 "Wednesday PM" (下午)。
5.  點擊 **"Add Unavailable Time"**。系統排班時會自動避開這些時段。

## 5. 產生排班表 (Generate Schedule)
當人員、假期與請假都設定完成後：
1.  點擊右上角的 **"Generate Schedule"** 按鈕。
2.  系統會依據公平的 "輪流 (Round Robin)" 機制自動填滿空缺。
    -   系統會遵守每月上限。
    -   系統會避開假期。
    -   系統會避開同仁的不便時段。

## 6. 查看統計 (Statistics)
-   **Monthly Summary (月統計)**: 日曆下方的表格顯示同仁 *本月份* 的值班數 (若達上限會顯示紅色)。
-   **Yearly Summary (年統計)**: 最下方的表格顯示 *整年度* 的累積值班數。
    -   **小撇步**: 將滑鼠游標停在數字上，可以看到詳細的月份分佈 (例如: "Jan: 2, Feb: 1")。

## 7. 清除年度資料 (Clear Year)
若您需要重新開始新的一年，或想刪除舊的測試資料：
1.  點擊上方標題旁的 **"Clear Year"** 按鈕。
2.  確認對話框。
3.  **警告**: 這將會刪除本年度**所有**已排定的值班紀錄。
